## Overview
OInbox is an **intelligent omnichannel inbox** designed to centralize customer conversations and automate lead handling using AI.

It enables businesses to manage, qualify, and route leads efficiently across messaging channels.

## Key Features
- Unified inbox for multiple channels
- AI-assisted lead qualification
- Automated follow-ups and tagging
- Human handoff and SLA control
- CRM and workflow integration

## Architecture
- Event-driven messaging pipeline
- AI-assisted classification and routing
- Multi-tenant SaaS architecture

## Tech Stack
- Frontend: TypeScript / Web UI
- Backend: Serverless APIs
- AI: NLP, LLM-based classification

## Status
Production-ready and actively used.
